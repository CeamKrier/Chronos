Chronos

The ultimate process tracker
with pomodoro timer

Chronos is a desktop process tracker that lets you analyze the daily active/inactive usage information of your applications.

And, it is open source!

Download for Windows Check Source Code

Features

Ultra-easy to use

Once you install the application, it will start to track your session automatically.

Chronos has ability to pause itself when you lock your screen and can resume when you come back.

The application will conclude the day's session at 12:00 a.m every day.

No distractions. Application can minimize itself to the system tray.

Customizable

You can decide whether to run Chronos on the system start-up.

Chronos comes with an integrated pomodoro timer which is fully customizable.

Whenever a pomodoro cycle completed, a notification will be sent to inform you.

You can reset the pomodoro timer by hovering and clicking on the pomodoro counter in the bottom of the screen.
